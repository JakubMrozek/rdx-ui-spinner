# rdx-ui-spinner

Spinner for react redux app.

## Install

```
npm install rdx-ui-spinner
```

## Usage

```javascript
import Spinner from 'rdx-ui-spinner'

//1. import default styles
import from 'rdx-ui-spinner/styles/spinner.scss'

//2. create component
<div>
  <Spinner/>
</div>

```
